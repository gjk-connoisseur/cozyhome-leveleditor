P5TextBox

Displays an editable text box. See: Using a text box control.

Variables

  • (int) absx

    Contains the absolute x position of the control. This field is read-only.

  • (int) absy

    Contains the absolute y position of the control. This field is read-only.

  • (boolean) acceptFloat

    Enables input of a float seperator ("."). Used with numOnly.

  • (boolean) anchorBottom

    Makes the control anchored to the bottom of the window. See: Anchors.

  • (boolean) anchorLeft

    Makes the control anchored to the left side of the window. See: Anchors.

  • (boolean) anchorRight

    Makes the control anchored to the right side of the window. See: Anchors.

  • (boolean) anchorTop

    Makes the control anchored to the bottom of the window. See: Anchors.

  • (boolean) blinkerVisible

    If true then the text box cursor is currenly visible. This field is read-only.

  • (int) borderWidth

    Gets or sets the width of the control border.

  • (int) cursorPos

    Gets or sets the position of the cursor (in chars).

  • (int) h

    Gets or sets the height of the control.

  • (boolean) hover

    If true then mouse is over this control. This field is read-only.

  • (boolean) isResizable

    If true then the control is resizable. This field is read-only.

  • (array of string) lines

    Cotains an array of all the lines currently in the text box. This field is read-only.

  • (int) longestLineLength

    Contains the length of the longest line currently in the text box (in pixels). This field is read-only.

  • (boolean) maxLength

    Gets or sets the text length limit of the text box. Set to -1 if the text length should be unlimited.

  • (boolean) multiline

    Enables the input of multiple lines.

  • (boolean) numOnly

    Limits the text box to only accept numbers.

  • (P5Container) parent

    Contains the parent container of the control. This field is read-only.

  • (int) selectionEnd

    The position at which the current text selection ends. Equal to -1 if there is no text selected. This field is read-only.

  • (int) selectionStart

    The position at which the current text selection starts. Equal to -1 if there is no text selected. This field is read-only.

  • (P5ControlStyle) style

    Contains the control style. See: Styling.

  • (boolean) textFits

    If true then the text fits into the control. This field is read-only.

  • (boolean) textSelected

    If true then there is text currently selected. This field is read-only.

  • (string) text

    Gets or sets the value of the text box.

  • (int) w

    Gets or sets the width of the control.

  • (P5Form) window

    Contains the parent window of the control. This field is read-only.

  • (string) type

    Contains the type of the object. See: Control types. This field is read-only.

  • (int) x

    Gets or sets the button x position.

  • (int) y

    Gets or sets the button y position.

Functions

cursorEnd()

Sends the cursor to the end of the text

cursorHome()

Sends the cursor to the start of the text

deselectText()

Deselects all the text

getSelectedText()

Returns the selected text value

keyPressed(k)

Simulates a key press

  • k - string value of the wanted key

pasteText(text)

Checks if the given string can be added into the text box and pastes it in the place of the cursor

  • text - text that should be pasted into the text box

removeSelectedText()

Removes the selected text from the text box

selectText(from, to)

Selects the text from one point to another

  • from - selection start
  • to - selection end

Events

See: Events

onDeactivate()

Called after the text box is deactivated.

onChange()

Called after the text box text is changed.

onClick()

Called after the mouse is released inside of the text box.
