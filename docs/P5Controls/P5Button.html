P5Button

Displays a button

Variables

  • (int) absx

    Contains the absolute x position of the control. This field is read-only.

  • (int) absy

    Contains the absolute y position of the control. This field is read-only.

  • (boolean) anchorBottom

    Makes the control anchored to the bottom of the window. See: Anchors.

  • (boolean) anchorLeft

    Makes the control anchored to the left side of the window. See: Anchors.

  • (boolean) anchorRight

    Makes the control anchored to the right side of the window. See: Anchors.

  • (boolean) anchorTop

    Makes the control anchored to the bottom of the window. See: Anchors.

  • (int) borderWidth

    Gets or sets the width of the control border.

  • (int) h

    Gets or sets the height of the control.

  • (boolean) hover

    If true then mouse is over this control. This field is read-only.

  • (image) image

    Gets or sets the image of the control.

  • (boolean) isResizable

    If true then the control is resizable. This field is read-only.

  • (P5Container) parent

    Contains the parent container of the control. This field is read-only.

  • (P5ControlStyle) style

    Contains the control style. See: Styling.

  • (string) text

    Gets or sets the text of the control.

  • (string) type

    Contains the type of the object. See: Control types. This field is read-only.

  • (int) w

    Gets or sets the width of the control.

  • (P5Form) window

    Contains the parent window of the control. This field is read-only.

  • (int) x

    Gets or sets the x position of the control.

  • (int) y

    Gets or sets the y position of the control.
