P5Container

Displays a container that can hold controls inside of it. A container is typically part of a form and should not be used seperately. See: Adding and managing controls.

Variables

  • (int) absx

    Contains the absolute x position of the control. This field is read-only.

  • (int) absy

    Contains the absolute y position of the control. This field is read-only.

  • (color) backColor

    Gets or sets the background color of the container.

  • (image) canvas

    Contains the canvas of the container. This field is read-only.

  • (array) controls

    Contains all the controls currently inside the container. This field is read-only.

  • (int) h

    Gets or sets the height of the control.

  • (int) mouseX

    Contains the mouse x position relative to the top left corner of the container. This field is read-only.

  • (int) mouseY

    Contains the mouse y position relative to the top left corner of the container. This field is read-only.

  • (P5Container) parent

    Contains the parent container of the control. This field is read-only.

  • (P5TextBox) selectedTextBox

    Contains the text box that is currently selected. The value is null when no text box is selected. This field is read-only.

  • (P5ControlStyle) style

    Contains the control style. See: Styling.

  • (P5TextBoxOverflow) textBoxOverflow

    Contains the text box overflow control. This field is read-only.

  • (P5ToolTip) toolTip

    Contains the tool tip control. This field is read-only.

  • (string) type

    Contains the type of the object. See: Control types. This field is read-only.

  • (int) w

    Gets or sets the width of the control.

  • (P5Form) window

    Contains the parent window of the control. This field is read-only.

  • (int) x

    Gets or sets the x position of the control.

  • (int) y

    Gets or sets the y position of the control.

Functions

addControl(ctrl)

Adds a given control to the container.

  • (ANY CONTROL) ctrl - Control that should be added to the container

controlAtPoint(x, y)

Returns a control that is at the given point. Returns null if there is no control at that point.

  • (int) x - x position of the point
  • (int) y - y position of the point

deactivateAllTextBoxes()

Deactivates all text box controls currently inside of the container.

removeAllControls()

Removes all the controls currently inside of the container.

getControlsOfType(type)

Returns all the controls of a given type that are currently in the container.

  • (string) type - Control type. See: Control types

removeControl(ctrl)

Removes a given control from the container.

  • (ANY CONTROL) ctrl - Control that should be removed from the container

Events

See: Events

onClick

Triggered when the mouse is released inside of the container.
