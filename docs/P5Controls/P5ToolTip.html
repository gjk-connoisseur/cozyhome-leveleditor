P5ToolTip

Displays a tool tip. A tool tip is typically part of a container and should not be used seperately. See: Using a tool tip control.

Variables

  • (int) h

    Contains the height of the control.

  • (boolean) isResizable

    If true then the control is resizable. This field is read-only.

  • (int) w

    Contains the width of the control.

  • (int) offsetX

    Gets or sets the distance from the cursor to the left side of the tool tip.

  • (int) offsetY

    Gets or sets the distance from the cursor to the top side of the tool tip.

  • (string) type

    Contains the type of the object. See: Control types. This field is read-only.

  • (int) x

    Contains the x position of the control.

  • (int) y

    Contains the y position control.

Functions

addToolTip(ctrl, tipText)

Adds a tooltip to the specified control

  • ctrl - the control that should have a tool tip
  • tipText - the text of the tool tip
