P5FormStyle

Enables styling forms. See: Styling.

Variables

  • (color) borderColor

    Gets or sets the color of a window border.

  • (color) ghostColor

    Gets or sets the color of a window ghost.

  • (color) inactiveTitleBarColor

    Gets or sets the color of an inactive window title bar.

  • (color) titleBarButtonColor

    Gets or sets the background color of a clicked title bar icon.

  • (color) titleBarColor

    Gets or sets the color of an active window title bar.

  • (color) titleBarFontColor

    Gets or sets the font color of a window title bar.

  • (int) windowAnimationLength

    Gets or sets the window animation length.

  • (color) windowShadowColor

    Gets or sets the color of a window shadow.

  • (int) windowShadowDistance

    Gets or sets the distance of a window shadow.
