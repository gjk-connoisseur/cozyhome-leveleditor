P5FormManager

Enables rendering and managing forms

Variables

  • (boolean) controlKeyPressed

    If true then the control key is currently pressed. This field is read-only.

  • (P5ControlStyle) controlStyle

    Gets or sets the current control style. See: Styling.

  • (string) font

    Gets or sets the the name of the font family used to draw text inside forms.

  • (int) fontLeading

    Gets or sets the the vertical spacing of the font used to draw text inside forms.

  • (int) fontSize

    Gets or sets the the size of the font used to draw text inside forms.

  • (array of P5Form) forms

    Contains all the forms currently shown.

  • (float) libVersion

    Contains the library version in the VERSION.YYMMDD format. This field is read-only.

  • (boolean) mouseIsPressed

    If true then a mouse button is currently pressed. This field is read-only.

  • (boolean) mouseIsReleased

    If true then a mouse button was released in the current frame. This field is read-only.

  • (P5Form) selectedWindow

    Contains the window that is currently on top. This field is read-only.

  • (boolean) shiftKeyPressed

    If true then the shift key is currently pressed. This field is read-only.

  • (P5FormStyle) windowStyle

    Gets or sets the current window style of all the children. See: Styling.

  • (boolean) windowMoveInProgress

    If true then a child window is being moved or resized. This field is read-only.

Functions

bringFormToFront(form)

Brings a given form to front

  • (P5Form) form - Form that should be brough to front

closeForm(form)

Closes a given form

  • (P5Form) form - Form that should be closed

(boolean) isOnTop(form)

Returns if a given form is currently on top

  • (P5Form) form - Form that should be checked

keyPressed()

Triggered when a key is pressed. The user must manually include this function in the default p5.js keyPressed() function.

keyReleased()

Triggered when a key is pressed. The user must manually include this function in the default p5.js keyReleased() function.

(boolean) pointInsideBounds(px, py, x, y, w, h)

Returns if a given point is inside given rectangle

  • (int) px, py - position of the point
  • (int) x, y - position of the rectangle
  • (int) w, h - size of the rectangle

(boolean) pointInsideTopWindow(x, y)

Returns if a given point is inside selected window

  • (int) x, y - position of the point

renderForms()

Renders all the open forms to the canvas

showForm(form)

Displays a given form

  • (P5Form) form - Form that should be displayed

Setup

var manager; function preload() { ... manager = new P5FormManager(); // !IMPORTANT! Instance the manager in the preload function ... // (otherwise images won't load properly) } function keyPressed() { ... manager.keyPressed(); // Add manager keyPressed function to the p5.js keyPressed event function ... } function keyReleased() { ... manager.keyReleased(); // Add manager keyReleased function to the p5.js keyReleased event function ... } function draw() { ... manager.renderForms(); // Render the forms ... }
