P5Forms

The p5forms.js file contains all the form-related classes

Variables

  • (image) BTN_CHECKED_ICON

    Contains the checked checkbox image.

  • (P5WindowStyle) DEFAULT_WINDOW_STYLE, (P5ControlStyle) DEFAULT_CONTROL_STYLE

    Contains the default window and control styles.

  • (image) MSGBOX_WARN_ICON, MSGBOX_INFO_ICON, MSGBOX_STOP_ICON, MSGBOX_QUESTION_ICON

    Contains the message box icons.

  • (image) WND_MAXIMIZE_ICON, WND_RESTORE_ICON, WND_CLOSE_ICON, WND_RESIZE_ICON, WND_MINIMIZE_ICON

    Contains the window title bar icons.

  • const (int) WND_STATE_NORMAL = 0, WND_STATE_MAXIMIZED = 1, WND_STATE_MINIMIZED = 2

    Contains the window state constants passed into the P5Form.setWindowState() function.
