P5Form

Displays a window. See: Adding and managing windows.

Variables

  • (int) absx

    Contains the absolute x position of the window. This field is read-only.

  • (int) absy

    Contains the absolute y position of the window. This field is read-only.

  • (int) borderWidth

    Gets or sets the window border width.

  • (P5Container) container

    Contains the container control for this window. This field is read-only.

  • (boolean) enableHelpLines

    Gets or sets the visibility of control bounds. See: Help lines.

  • (boolean) enableResizing

    Gets or sets the ability for the user to resize the window.

  • (boolean) enableDragging

    Gets or sets the ability for the user to move the window.

  • (int) h

    Gets or sets the height of the window.

  • (boolean) isOnTop

    If true then the window is on top and selected. This field is read-only.

  • (boolean) keepBehind

    If set to true, then the window will be kept under other windows.

  • (P5FormManager) manager

    Contains the manager of the form. This field is read-only.

  • (int) minh

    Gets or sets the minimum window height.

  • (int) minw

    Gets or sets the minimum window width.

  • (boolean) mouseIsPressed

    If true then the window is selected and it's not being moved by the user and the mouse is pressed. This field is read-only.

  • (boolean) mouseIsReleased

    If true then the window is selected and it's not being moved by the user and the mouse was released in this frame. This field is read-only.

  • (int) mouseX

    Contains the mouse x position relative to the x position of the window. This field is read-only.

  • (int) mouseY

    Contains the mouse y position relative to the y position of the window. This field is read-only.

  • (boolean) moveInProgress

    If true then the window is currently being moved by the user now. This field is read-only.

  • (P5Form) msgbox

    Gets or sets the child of a form.

  • (boolean) resizeInProgress

    If true then the window is currently being resized by the user now. This field is read-only.

  • (P5FormStyle) style

    Gets or sets the style of the particular form. For global use it's better to set the P5FormManager.windowStyle variable.

  • (string) title

    Gets or sets the window title.

  • (int) titleBarHeight

    Gets or sets the window title bar height.

  • (string) type

    Contains the type of the object. See: Control types. This field is read-only.

  • (int) w

    Gets or sets the width of the window.

  • (int) windowState

    Contains the current window state. For setting the window state see P5Form.setWindowState(). This field is read-only.

  • (int) x

    Gets or sets the x position of the window.

  • (int) y

    Gets or sets the y position of the window.

Functions

centerOnScreen()

Centers the window on screen.

removeWindowIcon(wndic)

Removes a given window icon from the title bar.

  • wndic - Window icon that should be removed WND_MAXIMIZE_ICON, WND_RESTORE_ICON, WND_CLOSE_ICON, WND_RESIZE_ICON, WND_MINIMIZE_ICON

setWindowState(targetWindowState)

Sets the state of the window to a given state.

  • targetWindowState - Target window state WND_STATE_NORMAL, WND_STATE_MAXIMIZED, WND_STATE_MINIMIZED

Events

See: Events

onClose()

Called before the window is closed by the manager.

onShow()

Called after the window is shown by the manager.

onRender()

Called after each window render.

onResize()

Called after window resize.
