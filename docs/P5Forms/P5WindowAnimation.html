P5WindowAnimation

Displays a window

Variables

  • (boolean) isPlaying

    If true then the animation is in progress.

Functions

playAnimation(ix, iy, iw, ih, tx, ty, tw, th, len)

Plays a window animation.

  • ix, iy - Initial rectangle position
  • iw, ih - Initial rectangle size
  • tx, ty - Target rectangle position
  • tw, th - Target rectangle size
  • len - Animation length

render()

Draw the animation on screen.
