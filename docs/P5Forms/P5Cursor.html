P5Cursor

Draws a custom mouse cursor

Variables

  • (image) image

    Gets or sets the cursor image.

Functions

render()

Renders the cursor on screen

Setup

var cur = new P5Cursor(); function setup() { ... noCursor(); // Hide the system mouse cursor ... } function draw() { ... cur.render(); // Render the p5forms cursor ... }
