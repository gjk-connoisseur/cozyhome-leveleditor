P5MsgBox

Displays a message box form. See: Using a message box.

Functions

showMsgBox(wnd, title, text, icon, buttons, callback)

Displays a message box form

  • (P5Form) wnd - The form that the message box should be displayed over
  • (string) title - The text that should be displayed on the message box titlebar
  • (string) text - The text that should be displayed inside the message box
  • (image) icon - MSGBOX_WARN_ICON, MSGBOX_INFO_ICON, MSGBOX_STOP_ICON, MSGBOX_QUESTION_ICON or any other image that should be displayed inside the message box
  • (Array) buttons - An array of strings containing the text of the buttons that should be displayed
  • (function) callback - A function that will be called after a message box button has been pressed. The text of the selected button is passed into the callback function as an argument.
