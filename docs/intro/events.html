Creating an interactive form

Paste the following code into your setup function

createCanvas(1280, 720, P2D); var form = new P5Form(); form.w = 200; form.h = 100; form.title = "Control events"; var label = new P5Label(); label.w = 200; label.text = "Don't press the button!"; var button = new P5Button(); button.y = 30; button.onClick = function() { // This is how you create an event handler form.container.backColor = "red"; label.text = "YOU PRESSED THE BUTTON! >:(" } form.container.addControl(label); form.container.addControl(button); formManager.showForm(form);
