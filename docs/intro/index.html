The basics

Create project folder. Your folder should contain the following files:

  • index.html

    Create an empty file

  • script.js

    Create an empty file

  • p5.js

    Copy this file from the p5.js library

  • p5controls.js

    Copy this file from the p5forms library

  • p5forms.js

    Copy this file from the p5forms library

  • p5forms_fonts.css

    Copy this file from the p5forms library

  • p5forms_data

    Copy this file from the p5forms library

Paste the following markup into the index.html file. <!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta http-equiv="X-UA-Compatible" content="IE=edge"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Project name goes here</title> <!--Import the p5.js library--> <script src="p5.js"></script> <!--Import the p5forms library--> <script src="p5forms.js"></script> <script src="p5controls.js"></script> <!--Import the script file--> <script src="script.js" defer></script> </head> <body></body> </html>

Creating and setting up a form manager

Paste the following code into your script.js file. // Create a global variable for the form manager var formManager; function preload() { // Create the form manager formManager = new P5FormManager(); } function setup() { // Create a drawing canvas createCanvas(1280, 720, P2D); } function draw() { // Clear the background background(255); // Render the forms formManager.renderForms(); } function keyPressed() { // p5.js key press event function // Notify the formManager formManager.keyPressed(); } function keyReleased() { // p5.js key release event function // Notify the formManager formManager.keyReleased(); }
