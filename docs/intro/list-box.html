Using a list box

Adding items

To add an item to a list box just use the following command.

myListBox.items.push("item");

Removing items

To remove an item from a list box use the following command.

myListBox.items.splice(i, 1);

where: i — the index of the item that should be removed

Getting the selected item

To get the selected item index use

var index = myListBox.selectedItem;

Removing all items

To remove all items from a list box use the following command. DO NOT use myListBox.items = [];. The items field should not be overwritten as it is a proxy to an array.

myListBox.items.splice(0, myListBox.items.length);
