Styling

See: P5FormStyle Documentation

See: P5ControlStyle Documentation

Create a window style

var myFormStyle = new P5FormStyle(); var background = color(0, 161, 255); var foreground = color(255, 70, 0); var shadow = color(0, 161, 255, 150); var title = color(255); myFormStyle.borderColor = foreground; myFormStyle.inactiveTitleBarColor = background; myFormStyle.titleBarButtonColor = background; myFormStyle.titleBarColor = background; myFormStyle.ghostColor = background; myFormStyle.titleBarFontColor = title; myFormStyle.windowShadowColor = shadow;

Create a control style

var myControlStyle = new P5ControlStyle(); var background = color(0, 161, 255); var foreground = color(255, 70, 0); myControlStyle.buttonClickedBorderColor = background; myControlStyle.listBoxSelectedColor = background; myControlStyle.checkBoxCheckColor = background; myControlStyle.listBoxBorderColor = background; myControlStyle.textBoxBorderColor = background; myControlStyle.buttonBorderColor = background; myControlStyle.buttonTextColor = foreground; myControlStyle.buttonClickedColor = foreground;

Apply the style

formManager.controlStyle = myControlStyle; formManager.windowStyle = myFormStyle;
