Adding and managing controls

Video tutorial

Creating and displaying a form with controls

var form, button, textBox; // preload(), setup()... function setup() { createCanvas(1280, 720, P2D); form = new P5Form(); form.w = 200; form.h = 200; form.title = "My form"; button = new P5Button(); button.y = 145; button.text = "Click me!"; textBox = new P5TextBox(); textBox.w = 185; textBox.text = "Type here..."; form.container.addControl(button); form.container.addControl(textBox); formManager.showForm(form); } // keyPressed(), keyReleased()...

Removing a control

Paste the following line into your browser console. The text box will be removed from the form. form.container.removeControl(textBox);

Removing all controls

Paste the following line into your browser console. All controls will be removed from the form. form.container.removeAllControls();
